Сайт для загрузки фотографий

Что использовалось: 
* Адаптивная верстка
* Флексбокс-верстка
* Гриды
* Анимация окон выполнена на JavaScript
* Плавность открытия pop-up выполнена на CSS 
* Использованы принципы ООП, созданы классы и разбито на модули
* Деструктуризация
* Слабое связывание классов
* Сборка Webpack
